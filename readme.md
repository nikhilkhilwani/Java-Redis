# **Working with Redis using Java**

***

This Project demonstrates different use cases of Redis Database using **core Java** Redis is an open-source, in-memory data structure store used as a database, cache, message broker, and streaming engine. It's known for its speed and flexibility, storing data in RAM for fast access. Redis supports various data structures like strings, hashes, lists, sets, and more, and offers features like replication, Lua scripting, and different levels of on-disk persistence

***

### Important
1. You should install **Redis** locally.
2. Start the Redis Server using **redis-server** command.
3. All the important Libraries are in the /lib of each folder. 
4. Each program has a comment in the last that shows how to compile and run that program.

***

### Index for the Projects

***

| #   | Title                  | Description                                                                                                                                                                      |
| --- | ---------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1   | SortedLeaderBoard.java | This program demonstrates the use of **Sorted Sets (zsets)** in Redis. Sorted Sets automatically sorts the elements in run-time. User should enter Name and Score of the players |
| 2   | /PubSub/               | This is a Publish/Subscribe implementation using Redis. /PubSub has 2 programs: Publisher.java and Subscriber.java. There can be n number of Subscribers but only one Publisher. Both Publisher and Subscriber should be on the same channel the receive the message. Run both programs simultaneously, publish using Publisher.java and you will receive the message on Subscriber.java. A .txt file is also generated name subscriber.txt which contains all the messages received by that subscriber |
